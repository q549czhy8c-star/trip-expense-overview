# 遊玩支出總覽

純前端多人分帳工具，適合旅遊、遊玩、聚餐或活動後快速整理支出。

## 功能

- 新增與管理參與人
- 為每筆支出指定先支付的人
- 支援平均分攤或自訂金額分攤
- 支援指定某人替其他人代付
- 以 HKD 統一計算總額與淨額
- 內建 RMB -> HKD 匯率轉換
- 顯示「誰欠誰」明細與最佳化還款建議
- 匯出 Excel 可開啟的總覽表格
- 自動把資料保存在瀏覽器本機儲存
- 使用純文字 SVG 網站圖示與 Web App Manifest，避免提交 PNG/ICO 等二進位圖檔

## 使用流程

1. 先加入所有參與人。
2. 設定預設匯率，例如 `1 RMB = 1.08 HKD`。
3. 逐筆新增支出，選擇支付人、貨幣與分攤方式。
4. 如果某人是替其他人先付，於該人的「由誰代付」欄位指定代付者。
5. 在頁面底部查看每人淨額、誰欠誰，以及最佳化還款建議。
6. 按下「匯出 Excel 總覽」下載表格。

## 示例情境

假設有 `阿明`、`阿欣`、`阿強` 三人出遊：

- `阿明` 先付 HKD 300 門票，三人平均分攤
- `阿欣` 先付 RMB 240 晚餐，三人平均分攤
- `阿強` 本來也要分一份門票，但由 `阿欣` 代付

系統會同時顯示：

- 每筆項目的分攤結果
- 累積後的「誰欠誰」關係
- 整體最少筆數的還款建議

## 技術說明

- 無框架靜態網站
- `index.html`: 頁面結構
- `styles.css`: 介面樣式
- `app.js`: 狀態管理、分帳計算、匯出功能

## 本地開啟

直接打開 `index.html` 即可使用，或用簡單靜態伺服器預覽：

```bash
python3 -m http.server 4173
```

然後開啟 `http://localhost:4173`。
